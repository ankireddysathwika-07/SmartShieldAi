# 🛡️ SmartShield AI

**SmartShield AI** is an AI-powered cybersecurity platform designed to protect users from modern digital threats. It helps detect phishing attacks, malicious URLs, spam messages, suspicious emails, and fraudulent QR codes using Artificial Intelligence, while providing real-time cybersecurity assistance through an AI chatbot.

---

## 🚀 Features

- 🔗 **URL Inspector**
  - Detects phishing and malicious websites.
  - Provides security risk analysis.

- 📧 **Email Analyzer**
  - Scans email content for phishing attempts.
  - Identifies suspicious attachments and links.

- 💬 **WhatsApp Spam Detection**
  - Detects spam and scam messages.
  - AI-based message classification.

- 📱 **QR Code Fraud Detection**
  - Scans QR codes before opening.
  - Warns users about malicious or fake QR codes.

- 🤖 **AI Cybersecurity Assistant**
  - Provides cybersecurity guidance.
  - Answers security-related questions in real time.

- 🚫 **AutoBlock Engine**
  - Automatically blocks detected malicious URLs and threats.
  - Prevents access to harmful content.

- 📊 **Security Dashboard**
  - Displays threat statistics and analysis.
  - Easy-to-use modern interface.

## ⚙️ Installation

### Clone Repository
```bash
git clone https://github.com/ankireddysathwika-07/SmartShield-AI.git
```

### Navigate to Project
```bash
cd SmartShield-AI
```

### Install Dependencies
```bash
npm install
```

### Start Development Server
```bash
npm run dev
```
